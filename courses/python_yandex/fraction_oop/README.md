\# Fraction OOP Practice



Учебный мини-проект по теме \*\*«Волшебные методы. Переопределение методов. Наследование»\*\* из курса \*\*Яндекс Handbook Python\*\*.



В проекте реализован класс `Fraction`, который представляет рациональные дроби и поддерживает основные арифметические операции, операции сравнения и работу с целыми числами.



\## Цель проекта



Цель проекта — закрепить навыки объектно-ориентированного программирования в Python и научиться переопределять поведение объектов через специальные методы.



В рамках проекта были отработаны:



\- создание собственного класса;

\- инкапсуляция данных;

\- работа с приватными полями;

\- переопределение строкового представления объекта;

\- переопределение арифметических операторов;

\- переопределение операторов сравнения;

\- поддержка операций между объектами класса и целыми числами;

\- автоматическое сокращение дробей.



\## Описание класса



Класс `Fraction` реализует рациональную дробь в виде числителя и знаменателя.



Дробь можно создать несколькими способами:



```python

Fraction(1, 3)

Fraction("1/3")

Fraction(2)

Fraction("2")

```



Класс автоматически сокращает дробь и корректно обрабатывает отрицательные значения.



Пример:



```python

a = Fraction(2, 6)

print(a)

```



Результат:



```text

1/3

```

```python

```

```python

```

\### Арифметические операции



Класс поддерживает:



\- сложение;

\- вычитание;

\- умножение;

\- деление;

\- операции с присваиванием.



Пример:



```python

a = Fraction(1, 3)

b = Fraction(1, 2)



print(a + b)

print(a - b)

print(a \* b)

print(a / b)

```



Результат:



```text

5/6

\-1/6

1/6

2/3

```



\### Операции с присваиванием



```python

a = Fraction(1, 3)



a += Fraction(1, 6)

print(a)



a \*= Fraction(2, 1)

print(a)

```



\### Операции сравнения



Класс поддерживает операторы:



\- `>`;

\- `<`;

\- `>=`;

\- `<=`;

\- `==`;

\- `!=`.



Пример:



```python

b = Fraction(1, 2)



print(a > b)

print(a < b)

```



Результат:



```text

```



\### Обратная дробь



Метод `reverse()` возвращает дробь, обратную текущей.



```python

print(a.reverse())

```



Результат:



```text

```

```python

print(2 + a)

print(2 - a)

print(2 / a)

```



Результат:



